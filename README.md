# AI-Powered-Resume-Analyzer-Interview-Preparation-Platform

## Overview

SmartHire AI is a full-stack web application designed to help students and job seekers improve their resumes and prepare for interviews using AI.

It analyzes resumes, detects missing skills, generates ATS-friendly suggestions, and creates professional PDF resumes dynamically.

It combines:

Resume Builder + Interview Coach + Career AI Assistant

into one practical system.

---

## Features

- Secure User Authentication using JWT
- Token Blacklisting for Secure Logout
- Resume Upload and Parsing
- AI-Powered Skill Extraction using Gemini API
- Skill Gap Detection and Improvement Suggestions
- ATS-Optimized Resume Generation
- Interview Preparation Reports
- Recent Reports Dashboard
- Dynamic Resume PDF Generation using Puppeteer
- Protected Routes and Session Management
